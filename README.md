# TypeScript - February 2026
Contains solutions from the SoftUni course.

<h3>About the course</h3>
<p>The "TypeScript" course focuses on mastering one of the most popular languages for building server-side and front-end applications. Created by Microsoft, TypeScript builds on JavaScript by adding static typing. The course covers working with the main types offered by the language, object-oriented principles using TypeScript—working with classes, inheritance, abstraction, interfaces, and generics. The training will also focus on modules and decorator functions. In the planned workshop, all the knowledge gained will be applied to building a complete TypeScript application.</p>
